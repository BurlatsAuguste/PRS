# PRS

Le but de ce projet est de designer et d'implémenter des server pour des clients donnés.
Nous ne disposons pas des codes sources des clients, uniquement des exécutables.

Le client va envoyer au serveur une requête de document, le server devra alors envoyer le document demandé au client.
Notre but est d'obtenir le meilleur débit possible lors de l'envoi de ce document.
